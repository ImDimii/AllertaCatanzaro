API Documentation

Get Weather Alert Information For Catanzaro

Retrieve weather alert information for both today and tomorrow.

Endpoint:

GET 94.176.233.185:3000/api/meteo

Parameters:

  • day (Optional): Specify 'today' or 'tomorrow' to get information for a specific day.

Example Requests:

// Get weather alert information for both days
GET 94.176.233.185:3000/api/meteo

// Get weather alert information for today only
GET 94.176.233.185:3000/api/meteo?day=today

// Get weather alert information for tomorrow only
GET 94.176.233.185:3000/api/meteo?day=tomorrow
